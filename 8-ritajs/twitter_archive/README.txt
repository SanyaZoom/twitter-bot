# Как пользоваться архивом Твиттера

С архивом Твиттера проще всего работать с помощью имеющегося в файле интерфейса браузера для архива. Для этого дважды щёлкните index.html в корневой папке и вы сможете посмотреть полную историю своих твитов прямо в браузере.

---

В папке `data` архив вашего Твиттера представлен в двух форматах: JSON и CSV, экспортированные по месяцам и годам.

* CSV — это универсальный формат, который можно импортировать во множество инструментов для работы с данными, например приложениями электронных таблиц, или просто обрабатывать с помощью языка программирования.

## JSON для разработчиков

* В экспорте JSON содержится полное представление ваших твитов, поскольку для их возвращения использовался API Твиттера версии 1.1. Более подробную информацию вы можете узнать на странице https://dev.twitter.com/docs/api/1.1.
* Экспорт JSON также используется в целях улучшения работы интерфейса браузера для архива (index.html).
* Для обработки экспорта с помощью универсального анализатора JSON в любом языке, удалите первую и последнюю строки в каждом файле.

Чтобы оставить отзыв, задать вопросы или поделиться своими идеями с другими разработчиками Твиттера, присоединяйтесь к обсуждению на форуме https://dev.twitter.com.
